# 🌿 L'Ombra del Basilico - Cucina Botanica

Benvenuti nel repository ufficiale de **L'Ombra del Basilico**, un sito web moderno ed elegante progettato per un ristorante d'eccellenza situato nel cuore di Foggia.

## ✨ Caratteristiche
- **Design Raffinato:** Un'estetica "botanica" che riflette l'identità del brand.
- **Totalmente Responsive:** Navigazione perfetta su smartphone, tablet e desktop.
- **Menu Digitale:** Consultazione rapida dei piatti e delle specialità.
- **Prenotazione Rapida:** Integrazione con WhatsApp per prenotazioni immediate.
